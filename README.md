# vscode-copy-without-markdown

This enables to copy text without markdown syntax. the keyboard shortcut is available only for markdown language.

## Installation

Search for `Copy without Markdown`

## Usage

- Mac OS : `Ctrl + Cmd + C`
- Windows, Linux : `Ctrl + Alt + C`
